🎯 A-6 Digital Tools Marketplace

    A modern, responsive, and interactive Digital Tools Marketplace UI built with React, Vite, and Tailwind CSS.
    This project demonstrates a full-fledged frontend workflow including product listing, cart functionality, dynamic UI, and user-friendly interactions.

    🌐 Live Demo & Repository
    🔗 Live Site: https://digitoolsplatformassi6.netlify.app/
    🔗 GitHub Repository: https://github.com/shahrierbiddut/Assi_6_Hero
    🚀 Project Overview

A-6 Digital Tools Marketplace is a fully responsive React-based web application that simulates a digital tools buying platform.

    Users can:

    Explore products with detailed descriptions
    Add/remove items in the cart dynamically
    Experience real-time updates in cart and notifications
    Navigate easily between products and cart views

    The project emphasizes clean UI, responsive design, and modern frontend best practices.

🎯 Project Motivation

    The motivation behind this project:

    Build a realistic e-commerce UI for digital products
    Practice React component architecture and state management
    Integrate dynamic JSON-based data loading
    Implement interactive user experience with modern tools like React-Toastify

🛠️ Technologies Used

    Notifications	React-Toastify
